# Smart Power Cut & Voltage Monitoring System

## Overview
A C-based monitoring system that detects power cuts, low voltage, high voltage, and normal operating conditions.

## Features
- Power Cut Detection
- Low Voltage Detection
- High Voltage Detection
- Continuous Monitoring
- Event Statistics

## Technologies Used
- C Programming
- VS Code
- GitHub
